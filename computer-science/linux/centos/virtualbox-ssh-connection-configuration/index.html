CentOS7 VirtualBox SSH连接配置

发表于2022-09-13本文字数719字阅读时长1 分钟

首页计算机科学LinuxCentOS

由于 CentOS 在 VirtualBox 中经常出现无法使用与主机共享粘贴板
可以考虑使用 SSH 来替代

# 安装 openssh-server

sudo yum install openssh-server

# 配置

sudo vim /etc/ssh/sshd_config

找到 Port 22 解注释，去掉前面的 #

找到 PermitRootLogin ，解注释，并且确认是 yes

# 开启 SSH 并设为启动项

systemctl start sshd.service
systemctl enable sshd.service

# 虚拟机端口映射

VirtualBox > 设备 > 网络 > 高级 > 端口转发
主机端口自定义

子系统 IP 通过 ifconfig 获取

其余按照图示配置即可

# SSH 连接

在主机的终端中运行

ssh -p 2222 stu@127.0.0.1

2222 为端口名称
stu 为用户名
视情况进行更改

# 最后

之后如果还需要建立 SSH 连接，先打开虚拟机，然后在主机终端中输入上面那一串内容即可

VirtualBoxCentOSLinuxSSH
